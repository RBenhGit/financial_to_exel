# Description: Create profitability ratio calculations including margin analysis, return ratios, and earnings-based metrics
# Details:
Develop profitability ratios using income statement and balance sheet data: Gross Profit Margin (Gross Profit / Revenue), Operating Profit Margin (Operating Income / Revenue), Net Profit Margin (Net Income / Revenue), EBITDA Margin (EBITDA / Revenue), Return on Assets (ROA = Net Income / Average Total Assets), Return on Equity (ROE = Net Income / Average Shareholders' Equity), Return on Invested Capital (ROIC = NOPAT / Invested Capital), Basic Earnings Per Share (Net Income / Weighted Average Shares), Diluted EPS calculations. Implement proper handling of preferred dividends and share count adjustments. Use trailing twelve months (TTM) and annual data with appropriate averaging for balance sheet items.
