# Status: done

# Description: Develop leverage and solvency ratios for debt analysis and financial risk assessment
# Details:
Build comprehensive leverage ratio calculations: Debt-to-Assets Ratio (Total Debt / Total Assets), Debt-to-Equity Ratio (Total Debt / Total Equity), Debt-to-Capital Ratio (Total Debt / (Total Debt + Total Equity)), Interest Coverage Ratio (EBIT / Interest Expense), Debt Service Coverage Ratio (Operating Income / Total Debt Service), Cash Coverage Ratio ((EBIT + Depreciation) / Interest Expense), Capitalization Ratio (Long-term Debt / (Long-term Debt + Shareholders' Equity)), Financial Leverage Multiplier (Total Assets / Total Equity). Implement proper debt classification and off-balance-sheet adjustments per current accounting standards (ASC 842 for leases).

# Test Strategy:
Test with highly leveraged companies and utilities. Verify against credit rating agency methodologies. Test lease capitalization calculations under ASC 842 standards.
