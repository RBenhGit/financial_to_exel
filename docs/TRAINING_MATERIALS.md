#### Topics Covered

##### 1.1 Financial Analysis Fundamentals (30 minutes)
**Free Cash Flow (FCF) Concepts:**
- **FCFE (Free Cash Flow to Equity)**: Cash available to shareholders
  - Formula: Operating Cash Flow - CapEx - Net Debt Payments
  - Interpretation: Positive FCFE indicates cash generation for shareholders

- **FCFF (Free Cash Flow to Firm)**: Cash available to all stakeholders
  - Formula: Operating Cash Flow - CapEx + Tax Shield on Interest
  - Interpretation: Total cash generated by business operations

- **Levered FCF**: FCF considering debt impact
  - Formula: Net Income + Depreciation - CapEx - Working Capital Change
  - Interpretation: Cash flow after debt service obligations

**Key Metrics:**
- Growth Rates: Year-over-year FCF changes
- FCF Yield: FCF / Market Capitalization
- FCF Margin: FCF / Revenue

##### 1.2 Application Navigation (20 minutes)
**Interface Overview:**
- Main navigation tabs (FCF Analysis, DCF, DDM, P/B, Portfolio, Watch Lists)
- Data input methods (Ticker symbol, Excel upload)

**Best Practices:**
- Start with ticker symbol input for beginners
- Use sample companies (AAPL, MSFT, GOOGL) for practice

#### Topics Covered

##### 2.1 DCF Valuation Theory (45 minutes)
**Core Concepts:**
- **Present Value**: Future cash flows discounted to today's value
- **Enterprise Value**: Total company value (debt + equity)
- **Terminal Value**: Value beyond forecast period
- **WACC**: Weighted Average Cost of Capital as discount rate

**DCF Formula:**
```
Enterprise Value = Σ(FCF_t / (1 + WACC)^t) + Terminal Value / (1 + WACC)^n
Equity Value = Enterprise Value - Net Debt
Value per Share = Equity Value / Shares Outstanding
```

**Key Assumptions:**
- **Discount Rate (WACC)**: Typically 8-12% for most companies
- **Terminal Growth Rate**: Long-term economy growth (2-4%)
- **Forecast Period**: Usually 5-10 years

##### 2.2 Hands-On Practice: DCF Analysis (90 minutes)
**Exercise 2: Microsoft (MSFT) DCF Valuation**

1. **Setup and Data Gathering (20 minutes)**
   - Navigate to DCF Analysis tab
   - Input ticker: MSFT
   - Review fetched data:
     - Revenue: $211B+ (recent)
     - FCF: $65B+ (recent)
     - Market Cap: Current market value
     - Debt levels and cash position

2. **Configure Assumptions (25 minutes)**
   - **Discount Rate**: Start with 10% (adjust based on company risk)
   - **Terminal Growth Rate**: Use 2.5% (US economy long-term growth)
   - **Revenue Growth**: Analyze historical trends (5-15% for tech)
   - **FCF Margins**: Review sustainability of current margins

3. **Run DCF Calculation (20 minutes)**
   - Execute valuation model
   - Review calculated components:
     - Present Value of FCFs (Years 1-10)
     - Terminal Value calculation
     - Enterprise Value
     - Equity Value per share

4. **Results Analysis (25 minutes)**
   - **Fair Value vs. Current Price**: Compare calculated fair value with market price
   - **Upside/Downside**: Calculate percentage difference
   - **Sensitivity Analysis**: Test impact of ±1% discount rate change
   - **Reasonableness Check**: Does result align with peer valuations?

**Expected Results (Illustrative):**
- Fair Value: $280-320 per share (will vary with market conditions)
- Current Price: Compare with real-time market price
- Valuation Range: Consider uncertainty in assumptions

##### 2.3 Dividend Discount Model (DDM) (45 minutes)
**DDM Theory:**
- Suitable for dividend-paying stocks
- Values company based on future dividend payments
- Three main variants: Gordon Growth, Two-Stage, Multi-Stage

**Exercise 3: Coca-Cola (KO) DDM Analysis**
1. **Dividend History Review (15 minutes)**
   - Navigate to DDM Analysis tab
   - Input ticker: KO
   - Review dividend history chart
   - Note dividend growth consistency

2. **Model Selection (10 minutes)**
   - Choose Gordon Growth Model (mature company)
   - Input current dividend: ~$1.76 per share
   - Input growth rate: ~3-5% (historical average)
   - Input required return: 8-10%

3. **Calculation and Interpretation (20 minutes)**
   - Run DDM calculation
   - Compare with current market price
   - Assess dividend sustainability
   - Review payout ratio (dividends/earnings)

#### Assessment
**Practical Exercise**: Perform DCF and DDM analysis on chosen company
- Must demonstrate proper assumption setting
- Must interpret results correctly
- Must identify potential valuation risks

**Exercise 5: Scenario Analysis**

3. **Risk Quantification (15 minutes)**

   - Assess downside protection

**Data Quality Framework:**

2. **Data Source Management (20 minutes)**

## 🏆 Best Practices for Training Success

### For Instructors
1. **Start with Concepts**: Ensure theoretical understanding before tool usage
2. **Use Real Examples**: Current market conditions and actual company data
3. **Encourage Experimentation**: Provide safe environment for learning
4. **Validate Understanding**: Regular comprehension checks and practical exercises
5. **Connect to Practice**: Link academic concepts to real-world applications

### For Students
1. **Practice Regularly**: Consistent application builds proficiency
2. **Start Simple**: Master basics before advancing to complex techniques
3. **Question Assumptions**: Understand the 'why' behind calculations
4. **Cross-Validate**: Compare results with external sources
5. **Document Learning**: Keep notes on insights and best practices

### Troubleshooting Common Issues
- **Data Quality Concerns**: When to trust vs. investigate further
- **Calculation Discrepancies**: Methodology differences vs. errors
- **Technology Issues**: Connectivity, performance, and compatibility
- **Interpretation Challenges**: Context and reasonableness assessment
